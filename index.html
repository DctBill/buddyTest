            ?user=
            时需要：填与机器人里存伙伴用的同一个 ID（一般为飞书

      构建：在 feishu-buddy-web 目录执行 npm run build，将本目录整包以
      HTTPS 部署（需带上 buddy-web.bundle.js）。?user=open_id 与机器人一致；?nonce=N 与
      重新孵化 对齐（机器人发链接）。若机器人已开只读 HTTP（BUDDY_HTTP_PORT），可加
      ?buddy_api=https://你的机器人域名 与可选 buddy_api_token=（与 BUDDY_HTTP_TOKEN 一致），页面会优先从接口取当前世代，可不依赖链接里的 nonce。抚摸约 2.5s 爱心 + 快速切帧；「说句闲话」模拟终端气泡（约 10s、末段变淡）。
